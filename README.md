## 使用说明

### 跟踪指数分级
- 将跟踪指数分级分成九级，相互独立。分别是：
    一级：同时满足交易量最大、费率最低中交易量最大、规模最大三者都是商务品；
    二级：同时满足交易量最大、费率最低中交易量最大是商务品、规模最大不是商务品；
    三级：同时满足交易量最大、规模最大是商务品、费率最低中交易量最大不是商务品；
    四级：同时满足费率最低中交易量最大、规模最大是商务品、交易量最大不是商务品；
    五级：仅满足交易量最大是商务品；
    六级：仅满足规模最大是商务品；
    七级：仅满足费率最低中交易量最大是商务品；
    八级：仅是商务品；
    九级：没有商务品。
